# HCAI 控制台接口清单（待填写）

> 用途：对接桌面端「HCAI 控制台」各板块时，以**浏览器 Network 实际抓包**为准填写。  
> 规则：**只写确认过的内容**；未抓到的项保持空白或写 `未确认`，不要凭猜测补路径/字段。  
> 敏感信息：token、完整 API Key、密码请打码后再粘贴。

---

## 填写说明

1. 打开 HCAI 网页控制台，登录后打开开发者工具 → **Network**。
2. 每点一个页面/按钮，在 Network 里找到对应 XHR/Fetch。
3. 复制：请求方法、完整 URL（含 query）、重要 Request Headers、Request Payload、Response Body。
4. 同一页面若有多个接口，复制下方「接口卡片」模板追加编号（如 `1.2`、`1.3`）。
5. 若一个接口服务多个板块，在「关联 UI」里写清，并在其它板块交叉引用即可。

## 附录：抓包粘贴小抄

在 Chrome/Safari Network 中：

1. 过滤：`Fetch/XHR`
2. 点选请求 → **Headers**：复制 Request URL、Method、Authorization 等
3. **Payload / Request**：复制 JSON
4. **Response**：复制 JSON（token 打码）
5. 若被 gzip/br 压缩，用 Response 面板的预览/格式化文本，不要贴乱码

填完后通知开发对接；未填项将不会实现为「猜的接口」。
